Colormap Curves

Adjust curves through different color spaces to create a scale from the colors below. Based on Gregor Aisch's Color Palette Helper with additional controls to help design diverging palettes.




Make sure to test the generated scale with a colorblind simulator.

github.com/PAIR-code/colormap

Bonus: open the Colormap Comparison tool in a new window to see your data transformed by the color scale created here.
